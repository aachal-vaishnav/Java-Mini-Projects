## üìå How to Run a Program

1. Make sure **Java (JDK)** is installed on your system.
2. Open terminal / command prompt.
3. Navigate to the folder containing the `.java` file.
4. Compile the program:
```bash
javac ProgramName.java
````

5. Run the program:

```bash
java ProgramName
```
